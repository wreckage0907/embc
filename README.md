# embc

## TO RUN  

1. Make BUILD Directory 
```bash
mkdir build
cd build
```
2. RUN the CMAKE FILE 
```bash
cmake ..
make
```

## HARDWARE CONNECTIONS
1. DHT22 Temperature Sensor
- Data - Pin 21
- VCC - Pin 36
- GND - Pin 38

2. MQ135 Air Quality Sensor
- AOUT - Pin 31
- VCC - Pin 36
- GND - Pin 38

3. OLED Display
- SCL - Pin 7
- SDA - Pin 6
- VCC - Pin 36
- GND - Pin 38
